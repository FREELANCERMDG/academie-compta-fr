# ANNEXE A — Checklists opérationnelles (regroupées)

> À imprimer / copier dans l'outil de suivi. Cocher avant chaque livraison.

---

## A1. Confidentialité (poste offshore)
- [ ] Poste dédié, antivirus actif, OS à jour
- [ ] Session verrouillée en cas d'absence
- [ ] Écran non visible des tiers
- [ ] Aucun fichier client sur messagerie/stockage personnel
- [ ] Accès via les seuls outils du cabinet
- [ ] Mots de passe forts + 2FA
- [ ] Wifi sécurisé

## A2. Ouverture / tenue de dossier mensuel
- [ ] Relevés bancaires complets reçus
- [ ] Pièces achats/ventes du mois reçues
- [ ] Pièces nommées selon la convention
- [ ] Doublons contrôlés
- [ ] Factures non conformes signalées
- [ ] Pièces manquantes listées et relancées
- [ ] Régime TVA + régime fiscal vérifiés (dossier permanent)

## A3. Saisie (par pièce)
- [ ] Pièce conforme et lisible
- [ ] Bon journal (AC/VT/BQ/CA/OD)
- [ ] Bon tiers (401/411)
- [ ] Bon compte de charge/produit
- [ ] Bon taux + bon compte de TVA
- [ ] HT + TVA = TTC (recalcul)
- [ ] Pas de doublon
- [ ] Commentaire si opération inhabituelle

## A4. Rapprochement bancaire
- [ ] Relevé complet, solde initial = solde final M-1
- [ ] Toutes les lignes pointées
- [ ] Frais/agios saisis
- [ ] Prélèvements/virements saisis
- [ ] Doublons supprimés
- [ ] Virements internes via 580 (= 0)
- [ ] État de rapprochement équilibré (écart 0 ou justifié)

## A5. Lettrage
- [ ] Lettrage automatique lancé
- [ ] Règlements partiels suivis
- [ ] Avoirs lettrés
- [ ] Écarts soldés (658/758/627)
- [ ] 411 créditeurs / 401 débiteurs analysés
- [ ] Créances anciennes signalées
- [ ] Balance âgée éditée
- [ ] 401/411 justifiés

## A6. Apurement du 471
- [ ] Grand livre 471 édité
- [ ] Chaque ligne commentée
- [ ] Pièces recherchées
- [ ] Imputations possibles passées
- [ ] Relances envoyées pour le reste
- [ ] 580 = 0, 455 justifié
- [ ] 471 ramené à 0 (ou restes transmis)

## A7. OD de paie
- [ ] Journal de paie reçu
- [ ] OD équilibrée (641+645 = 421+431+437)
- [ ] Net (421) = virements salaires
- [ ] Cotisations = DSN = prélèvements
- [ ] 421/431/437 soldés après paiement
- [ ] Écarts justifiés

## A8. Immobilisations
- [ ] Distinction charge/immo (seuil 500 € HT)
- [ ] TVA correcte (non déductible sur VP)
- [ ] Date de mise en service identifiée
- [ ] Durée conforme à la politique cabinet
- [ ] Prorata temporis 1re année
- [ ] Dotation passée (6811/28x)
- [ ] Fichier immo = compta (21/20/28)
- [ ] Cessions traitées (775/675)

## A9. Révision par cycle
- [ ] Trésorerie : rapprochements OK, 580 = 0, caisse ≥ 0
- [ ] Fournisseurs : 401 lettré/justifié
- [ ] Clients : 411 lettré/justifié
- [ ] Personnel : OD = bulletins = DSN = paiements
- [ ] Fiscal : comptes TVA cohérents
- [ ] Immobilisations : fichier = compta
- [ ] Emprunts : 164 = capital restant dû
- [ ] Capitaux : 455 justifié, résultat N-1 affecté
- [ ] 471/467 soldés ou justifiés
- [ ] Feuilles de travail rédigées

## A10. Préparation des éléments fiscaux
- [ ] Régime fiscal identifié (IR/IS, BIC/BNC, micro/réel)
- [ ] Comptes révisés et justifiés
- [ ] TVA de l'exercice bouclée
- [ ] Immobilisations à jour
- [ ] Provisions/dépréciations recensées (validation EC)
- [ ] Comptes courants d'associés justifiés
- [ ] Réintégrations/déductions signalées (pour décision EC)
- [ ] Tableaux de liasse pré-remplis
- [ ] Dossier transmis sans valider à la place de l'EC

## A11. Clôture / bilan
- [ ] Cut-off : FNP, FAE, CCA, PCA passées
- [ ] Stocks constatés (variation)
- [ ] Amortissements passés
- [ ] Provisions proposées (validation EC)
- [ ] TVA de clôture contrôlée
- [ ] Rapprochements de clôture OK
- [ ] Emprunts justifiés
- [ ] Paie justifiée
- [ ] Capitaux : résultat N-1 affecté
- [ ] 471/580 = 0
- [ ] Dossier bilan complet

## A12. ✅ QUALITÉ avant validation (à joindre à chaque livraison)
- [ ] Rapprochement bancaire bouclé, 512 justifié
- [ ] TVA contrôlée (taux, assurance/VP exclus, encaissements)
- [ ] 471 et 580 soldés
- [ ] Lettrage 401/411 réalisé, écarts soldés
- [ ] Aucun doublon
- [ ] Caisse 53 ≥ 0, 455 justifié
- [ ] Soldes anormaux expliqués
- [ ] Anomalies documentées
- [ ] Pièces présentes / non-conformités signalées
- [ ] Points en suspens listés pour le chef de mission

## A13. Organisation quotidienne (offshore)
- [ ] To-do priorisée du jour
- [ ] Échéances TVA de la semaine repérées
- [ ] Blocages remontés immédiatement
- [ ] Reporting de fin de journée envoyé
- [ ] Tableau de bord mis à jour
- [ ] Créneau de recouvrement FR respecté

## A14. Avant d'envoyer un mail
- [ ] Objet clair avec [CLIENT]
- [ ] Demande précise + action attendue + délai
- [ ] Contexte / solution fournis
- [ ] Pièces jointes nommées
- [ ] Pas de fautes, ton professionnel
- [ ] Pas d'info confidentielle en canal non sécurisé
