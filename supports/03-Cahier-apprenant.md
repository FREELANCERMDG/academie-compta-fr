## 6. Glossaire express
| Terme | Définition courte |
|---|---|
| CA3 / CA12 | Déclarations de TVA (réel normal mensuel / réel simplifié annuel) |
| Cut-off | Rattachement des charges/produits au bon exercice |
| FNP / FAE | Facture non parvenue / facture à établir |
| CCA / PCA | Charge / produit constaté d'avance |
| Lettrage | Association facture ↔ règlement |
| Rapprochement | Vérifier 512 comptable = relevé bancaire |
| Apurement | Solder un compte d'attente (471) |
| VNC | Valeur nette comptable (immo) |
| Autoliquidation | TVA collectée ET déduite par l'acheteur |
| Liasse fiscale | Tableaux fiscaux transmis avec la déclaration de résultat |
| DSN | Déclaration sociale nominative (paie) |
| Taux de retour | % de dossiers renvoyés pour correction (KPI qualité) |
