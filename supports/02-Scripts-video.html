Scripts vidéo (narration) — 20 modules

Pour enregistrer les vidéos de la formation (voix off + partage d'écran). Durée cible : 8–15 min/module. Structure : Accroche → Objectif → Démonstration → Points clés → Transition. Les passages [ÉCRAN : …] indiquent ce qu'on montre.

Vidéo 1 — Environnement comptable français

Accroche : « Avant de saisir la moindre écriture, il faut comprendre pour qui on travaille et ce qu'on attend de vous. »
Objectif : situer le collaborateur offshore dans la chaîne du cabinet.
[ÉCRAN : schéma cabinet → expert-comptable → chef de mission → collaborateur]
Points clés : différence saisie/tenue/révision/supervision/bilan ; dossier permanent vs annuel ; obligations comptables ; confidentialité/RGPD = éliminatoire ; les 4 piliers (rigueur, fiabilité, anticipation, traçabilité).
Message clé : « Vous préparez et proposez ; l'expert-comptable décide et signe. »
Transition : « Maintenant, parlons la langue des comptes : le PCG. »

Vidéo 2 — Bases de comptabilité (PCG)

Accroche : « Toute la comptabilité tient en une règle : débit = crédit. »
[ÉCRAN : tableau actif/passif/charges/produits ; passage d'une écriture d'achat]
Points clés : partie double ; journal→grand livre→balance→bilan/résultat ; les 7 classes ; comptes incontournables (401, 411, 512, 445, 6, 7, 2, 164, 641/645, 471, 455, 53).
Démo : saisir un achat, une vente, un encaissement.
Transition : « Avant de saisir, il faut un dossier propre. »

Vidéo 3 — Organiser un dossier externalisé

Accroche : « Un dossier bien rangé, c'est la moitié du travail. »
[ÉCRAN : arborescence GED + nommage AAAA-MM-JJ_TYPE_TIERS_MONTANT]
Points clés : réception/classement/nommage ; pièces manquantes ; doublons ; factures non conformes ; tableau de suivi.
Transition : « Place à la saisie. »

Vidéo 4 — Saisie comptable pratique

Accroche : « Le bon compte, le bon journal, la bonne TVA. »
[ÉCRAN : Pennylane/ACD — saisie achats, ventes, banque]
Points clés : schémas d'écritures ; imputation des charges courantes ; cas TVA (sans, autoliquidation, intracom, import/export) ; cas particuliers.
Démo : lot de factures variées.
Transition : « La TVA mérite son propre module. »

Vidéo 5 — TVA française

Accroche : « La TVA, c'est l'argent de l'État qui transite par l'entreprise. »
[ÉCRAN : mécanique collectée/déductible/à décaisser ; CA3]
Points clés : taux 2026 ; débits vs encaissements ; régimes et seuils 2026 ; CA3/CA12 ; non déductible (assurance, VP) ; contrôles avant déclaration.
Démo : préparer une CA3.
Message clé : « On prépare, le chef de mission valide et télétransmet. »

Vidéo 6 — Rapprochement bancaire

Accroche : « Si la banque est juste, l'essentiel de la compta est juste. »
[ÉCRAN : relevé vs grand livre 512, pointage]
Points clés : méthode de pointage ; écarts types ; recherche par montant/libellé ; virements internes (580) ; ne jamais forcer un écart.
Démo : rapprochement complet.

Vidéo 7 — Lettrage clients/fournisseurs

Accroche : « Lettrer, c'est savoir qui doit quoi. »
[ÉCRAN : grand livre 401/411, lettrage manuel d'un avoir]
Points clés : auto vs manuel ; écarts/avoirs/règlements partiels ; 411 créditeur, 401 débiteur ; balance âgée.

Vidéo 8 — Comptes d'attente et sensibles

Accroche : « Le 471 est une salle d'attente, pas une poubelle. »
[ÉCRAN : grand livre 471, arbre de décision]
Points clés : quand imputer/attendre/demander ; 467, 455 (alerte débiteur), 580 ; documenter une anomalie ; apurer avant clôture.

Vidéo 9 — Paie et écritures sociales

Accroche : « On comptabilise le journal de paie, pas chaque bulletin. »
[ÉCRAN : journal de paie Silae → OD]
Points clés : 641/645/421/431/437 ; repères 2026 (SMIC, PMSS 4 005 €) ; contrôle OD = bulletins = DSN = paiements.
Démo : OD de paie équilibrée.
