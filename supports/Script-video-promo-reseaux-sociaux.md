## 🤖 Outils IA recommandés (script + images → vidéo)
- **HeyGen** ou **Synthesia** : avatar qui parle + voix off (idéal si vous voulez un présentateur). Collez le texte voix off, choisissez une voix FR, ajoutez vos images.
- **InVideo AI** ou **Pictory** : transforment un script en vidéo montée automatiquement (B‑roll + sous‑titres + voix).
- **CapCut** (gratuit) : montage final, sous‑titres auto FR, musique, format 9:16.
- **ElevenLabs** : voix off FR très naturelle (si l'outil vidéo n'a pas une bonne voix).

**Méthode rapide** : 1) Voix off via ElevenLabs → 2) Images via Leonardo/Midjourney → 3) Montage + sous‑titres dans CapCut (modèle 9:16) → 4) Export 1080×1920.
