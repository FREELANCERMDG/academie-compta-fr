# 2.12 — Figer la comptabilité sur Pennylane

> 🎥 Démonstration — comment **figer (verrouiller) la comptabilité** dans Pennylane.

## Ce que montre la vidéo
Comment **figer la comptabilité** à une date donnée : une fois figée, **plus aucune écriture** ne peut être ajoutée, modifiée ou supprimée **avant cette date**.

## Pourquoi figer la comptabilité ?
- **Sécuriser** une période déjà déclarée (TVA, clôture) : on évite toute modification accidentelle.
- **Fiabiliser la liasse fiscale** : les calculs reposent sur des écritures **stables**.
- **Obligation légale** : une comptabilité doit être **intangible** une fois la période close.

## Les étapes clés
1. Ouvrir **Paramètres → Informations entreprise → Tenue comptable** (ou le menu dédié « Figer la comptabilité »).
2. Indiquer la **date jusqu'à laquelle** figer (souvent la **date de clôture** ou la fin d'une période de TVA).
3. **Valider** : toutes les écritures antérieures à cette date sont **verrouillées**.
4. Pour corriger une période figée → passer une **écriture sur une période ouverte** (OD de correction), sans toucher au passé.

## À retenir (réflexes cabinet)
- On **fige après chaque déclaration de TVA** et **à la clôture** : c'est une bonne pratique de contrôle interne.
- Figer **avant** de préparer la liasse fiscale garantit des **chiffres stables**.
- Une période figée se corrige **par une nouvelle écriture** sur une période ouverte, jamais en modifiant l'historique.
