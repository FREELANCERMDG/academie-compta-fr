# MODULE 17 — Qualité, erreurs fréquentes et contrôle interne

**Niveau :** Transversal · **Durée estimée :** 5–6 h · **Prérequis :** Modules 4 à 13

---

## 🎯 Objectifs pédagogiques
- Connaître les erreurs les plus fréquentes et savoir les éviter.
- Appliquer une méthode de contrôle qualité avant validation.

## 🧩 Compétences
- Auto-contrôler un dossier avant livraison.
- Réduire son taux de retour.

---

## 1. Les erreurs fréquentes (et comment les éviter)

| Erreur | Cause typique | Prévention |
|---|---|---|
| **Erreur de TVA** | Mauvais taux, TVA sur assurance/VP, encaissements oubliés | Contrôle TVA systématique (Module 5) |
| **Mauvais compte comptable** | Imputation par habitude, OCR non vérifié | Vérifier la nature réelle de la dépense |
| **Doublons** | Import + saisie manuelle | Tri par tiers/montant/n° facture |
| **Absence de justificatif** | Pièce manquante saisie quand même | 471 + relance, jamais d'invention |
| **Mauvais lettrage** | Lettrage auto sur écart | Lettrage manuel des écarts |
| **Solde bancaire non rapproché** | Rapprochement bâclé | Égalité 512 = relevé prouvée |
| **471 non apuré** | Opérations laissées en attente | Apurement avant livraison |
| **Charges personnelles en société** | Facture au nom du dirigeant | Signaler, ne pas déduire la TVA |
| **Mauvaise affectation client/fournisseur** | Tiers homonymes | Vérifier le bon compte tiers |
| **Confusion TTC / HT / TVA** | Calcul à l'envers | Recalculer HT + TVA = TTC |
| **Erreurs de paie** | OD ≠ journal de paie | Rapprocher OD = bulletins = DSN = paiements |
| **Anomalie sans commentaire** | Manque de traçabilité | Commenter chaque opération inhabituelle |

---

## 2. Le principe du contrôle interne offshore
- **Auto-contrôle** : le collaborateur contrôle son propre travail (checklist qualité).
- **Contrôle croisé** : si possible, un second regard sur les dossiers sensibles.
- **Supervision** : le chef de mission valide selon le niveau d'autonomie.
- **Documentation** : chaque contrôle laisse une trace (feuille de travail, checklist cochée).

---

## 3. La méthode de contrôle qualité (avant de dire « terminé »)
Procédure en 8 points :
1. **Banque rapprochée** : 512 = relevé (justifié).
2. **TVA cohérente** : collectée/déductible recalculées, pas d'erreur de taux.
3. **471 / 580 = 0** : comptes d'attente apurés.
4. **Lettrage fait** : 401/411 lettrés, écarts soldés.
5. **Doublons vérifiés** : aucun.
6. **Comptes anormaux contrôlés** : 53 ≥ 0, 455 justifié, pas de sens anormal injustifié.
7. **Anomalies commentées** : tout point en suspens documenté.
8. **Pièces** : justificatifs présents, factures non conformes signalées.

## ✅ Checklist qualité avant validation (à joindre à chaque livraison)
- [ ] Rapprochement bancaire bouclé et solde 512 justifié
- [ ] TVA contrôlée (taux, assurance/VP exclus, encaissements)
- [ ] 471 et 580 soldés
- [ ] Lettrage 401/411 réalisé, écarts soldés
- [ ] Aucun doublon
- [ ] Caisse 53 ≥ 0, 455 justifié
- [ ] Soldes au sens anormal expliqués
- [ ] Anomalies documentées (commentaires)
- [ ] Pièces présentes / non conformités signalées
- [ ] Points en suspens listés pour le chef de mission

## 4. Tenir son « carnet d'erreurs »
Chaque retour du cabinet = une ligne dans un fichier personnel :
| Date | Dossier | Erreur | Cause | Correction | Action préventive |
|---|---|---|---|---|---|

> C'est l'outil n°1 de progression : on ne refait pas deux fois la même erreur.

---

## 🧪 Cas pratique 17.1 — Trouver les erreurs
Sur un dossier livré, on observe : 471 = 1 500 D ; TVA déduite sur une facture d'assurance ; compte 53 = −30 ; une facture EDF saisie deux fois ; un 411 créditeur de 200 € non commenté. Listez les corrections.

### ✔️ Correction 17.1
1. **471 = 1 500** → apurer (imputer / relancer) avant livraison.
2. **TVA sur assurance** → annuler la TVA déduite (assurance exonérée).
3. **53 = −30** → corriger (caisse négative impossible).
4. **Doublon EDF** → supprimer une écriture.
5. **411 créditeur 200** → analyser (trop-payé) et **commenter** / reclasser.
**Conclusion :** dossier **non conforme** pour livraison tant que ces points ne sont pas réglés.
