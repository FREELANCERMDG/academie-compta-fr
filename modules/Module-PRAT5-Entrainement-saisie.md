# 3.12 — Entraînement : matrice de saisie (interactif)

À vous de jouer ! Pour chaque énoncé, **saisissez l'écriture** dans la grille (compte, libellé, débit, crédit), puis cliquez **« Vérifier mon écriture »**. Le système contrôle l'**équilibre** (débit = crédit) et l'**imputation**. Bloqué ? **« Voir la correction »**.

> 💡 Astuces : utilisez **« ➕ Ajouter une ligne »** si besoin · les montants s'écrivent `1 200` ou `1200,00` · le compte peut être saisi en abrégé (ex. `607`, `401`, `44566`).
