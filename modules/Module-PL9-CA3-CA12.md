# 2.9 — Formulaire CA3 / CA12 sur Pennylane

> 🎥 Démonstration — les **nouveaux formulaires de TVA CA3 / CA12** (mise à jour septembre 2025) dans Pennylane.

## Ce que montre la vidéo
Comment **remplir et générer** la déclaration de TVA dans Pennylane avec les **nouveaux formulaires** :
- **CA3** : déclaration **mensuelle ou trimestrielle** (régime réel normal) ;
- **CA12** : déclaration **annuelle** (régime réel simplifié).

## Rappel : CA3 ou CA12 ?
| Formulaire | Régime | Périodicité |
|---|---|---|
| **CA3** | Réel normal | **Mensuelle** (ou trimestrielle si TVA < 4 000 €/an) |
| **CA12** | Réel simplifié (RSI) | **Annuelle** + 2 acomptes (juillet, décembre) |

## Les étapes clés
1. S'assurer que la saisie est **complète** et que le **cadrage TVA** est fait (cf. leçon 2.8).
2. Ouvrir **Fiscalité → TVA** : Pennylane **pré‑remplit** le formulaire (CA3 ou CA12) à partir des écritures.
3. Contrôler chaque **cellule / cadre** du formulaire (bases par taux, TVA collectée, TVA déductible).
4. Vérifier le **solde** (TVA à payer ou crédit de TVA) puis **valider / télédéclarer**.

## À retenir (réflexes cabinet)
- Le formulaire est **pré‑rempli** par l'outil, mais le **contrôle reste indispensable** (cadrage avant validation).
- **CA3 = réel normal** (mensuel/trimestriel) · **CA12 = réel simplifié** (annuel + acomptes).
- Conserver le **détail justificatif** de chaque déclaration dans le dossier.
