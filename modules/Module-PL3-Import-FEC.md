## Ce que montre la vidéo
L'**import d'un FEC** (Fichier des Écritures Comptables) pour **reprendre une comptabilité existante** dans Pennylane.

## Rappel : qu'est‑ce que le FEC ?
Fichier **normalisé et obligatoire** (art. **A.47 A‑1** du LPF) qui contient **toutes les écritures** d'un exercice. C'est le format standard pour **migrer** une compta d'un logiciel vers un autre.

## Les étapes clés
1. **Récupérer le FEC** de l'ancien logiciel (fichier texte délimité `.txt`).
2. Dans Pennylane : lancer l'**import du FEC** (menu d'import / reprise de dossier).
3. **Contrôles après import** : la **balance reprise** doit être identique à la balance d'origine.
4. Vérifier les **comptes de tiers** (401/411), les **à‑nouveaux** et le **plan comptable**.

## À retenir (réflexes cabinet)
- Le FEC garantit une **reprise fidèle** sans ressaisie manuelle.
- **Toujours rapprocher la balance** après l'import (totaux débit = crédit, soldes identiques).
- En cas d'écart : vérifier l'**exercice**, l'**encodage** du fichier et les comptes manquants.
