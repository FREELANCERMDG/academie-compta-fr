# 3.10 — Réviser un dossier : justifier les comptes par cycle (pratique)

**Réviser**, c'est **justifier chaque solde** de la balance avec une **preuve**. Un compte est « justifié » quand on peut répondre : *« d'où vient ce solde, et par quel document je le prouve ? »*

> 🗂️ **Le dossier de travail** : pour chaque cycle, une **feuille de révision** = le solde, le justificatif, les contrôles et les écritures de régularisation. C'est ce que le chef de mission / l'expert‑comptable contrôle.

---

## 1) Cycle TRÉSORERIE — comptes 512 / 530 / 580
- **Justificatif** : relevé bancaire de clôture + **rapprochement bancaire**.
- **Contrôles** : solde comptable + écritures en rapprochement = solde relevé ; caisse (53) **jamais créditrice** ; virements internes (580) soldés à 0.
- **Régul typiques** : frais bancaires, agios, écarts de rapprochement.

## 2) Cycle CLIENTS — 411 / 416 / 418 / 491
- **Justificatif** : **balance âgée** clients + grand livre lettré.
- **Contrôles** : créances anciennes → relance ou **client douteux (416)** + **dépréciation (491)** ; **FAE (418)** pour les ventes non encore facturées.
- **Régul** : créances douteuses, FAE de clôture.

## 3) Cycle FOURNISSEURS — 401 / 408 / 409
- **Justificatif** : balance âgée fournisseurs + grand livre lettré.
- **Contrôles** : factures non parvenues → **FNP (408)** ; acomptes versés en **409** ; soldes débiteurs anormaux à analyser.

## 4) Cycle TVA & FISCAL — 4456 / 4457 / 44551 / 444
- **Justificatif** : **cadrage TVA** (TVA des comptes vs déclarations CA3) + dernière CA3.
- **Contrôles** : TVA collectée = CA × taux ; TVA déductible cohérente avec les achats ; **44551 (TVA à décaisser)** = montant réellement payé ; **444 (IS)** = acomptes + solde.

## 5) Cycle IMMOBILISATIONS — 20 / 21 / 28
- **Justificatif** : **tableau des immobilisations** et des amortissements.
- **Contrôles** : total fiches = comptes 21 ; **dotation de l'exercice (6811/28)** calculée ; **sorties/cessions** traitées (675/775, reprise amortissements).

## 6) Cycle PERSONNEL & SOCIAL — 42 / 43
- **Justificatif** : **journal de paie**, bordereaux de charges (DSN), états Silae.
- **Contrôles** : 421 (salaires à payer) = nets de décembre ; 43x (organismes sociaux) = cotisations dues ; **congés payés (provision)** si applicable ; 425 (notes de frais) soldé.

## 7) Cycle CAPITAUX & EMPRUNTS — 10 / 11 / 12 / 16
- **Justificatif** : statuts/PV (capital), **tableau d'amortissement de l'emprunt**.
- **Contrôles** : 164 = capital restant dû au 31/12 ; **intérêts courus (1688)** rattachés ; report à nouveau (11) cohérent avec l'affectation du résultat N‑1.

## 8) Cycle COMPTES D'ATTENTE — 471 / 472 / 47
- **Règle d'or** : un compte d'attente **doit être soldé à 0** à la clôture.
- **Contrôles** : analyser chaque ligne, retrouver la bonne imputation, **vider le 471**.

## 9) Cycle CHARGES & PRODUITS — classes 6 et 7
- **Justificatif** : grand livre + **cut‑off** (CCA / PCA pour rattacher au bon exercice).
- **Contrôles** : cohérence d'un mois à l'autre (pas de trou ni de doublon), charges non déductibles repérées pour la fiscalité.

---

## ✅ « Le compte est justifié quand… »
- j'ai le **document** qui prouve le solde,
- les **écritures de régularisation** sont passées,
- la **feuille de révision** est remplie et **datée**,
- je peux **expliquer le solde en une phrase** au chef de mission.

## 🧪 Mini‑exercice
Pour chacun, dites **avec quel justificatif** vous prouvez le solde : `512`, `411`, `44566`, `21`, `421`, `471`.
*(Réponses : relevé+rapprochement · balance âgée+lettrage · cadrage TVA · tableau immo · journal de paie · doit être à 0.)*
