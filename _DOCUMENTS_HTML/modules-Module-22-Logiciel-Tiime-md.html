MODULE 22 — Logiciel : Tiime (pré-compta TPE & cabinet)

Niveau : Intermédiaire · Durée estimée : 5–6 h (pratique) · Prérequis : Modules 4–7
Type : Formation logiciel — très utilisé pour les TPE/indépendants suivis par les cabinets.

🎯 Objectifs pédagogiques

  • Comprendre l'écosystème Tiime (côté client et côté cabinet).
  • Valider les écritures pré-catégorisées, associer les justificatifs, gérer les notes de frais.
  • Exporter proprement vers l'outil de production du cabinet.

🧩 Compétences

  • Contrôler/corriger la pré-catégorisation automatique.
  • Tenir un dossier Tiime propre et exportable.

1. À quoi sert Tiime

Suite française orientée TPE/freelancers, articulée avec le cabinet :

  • Tiime (app client) : compte bancaire pro connecté, facturation, justificatifs (photo), notes de frais, suivi de trésorerie.
  • Tiime côté cabinet (comptabilité) : récupération des transactions et pièces pré-catégorisées, contrôle, puis export vers le logiciel de production (ou tenue directe).

Force de Tiime : la pré-comptabilité automatique (rapprochement transaction ↔ justificatif, catégorisation suggérée).

2. Avant de produire : vérifications

  • Régime TVA et type d'activité (BIC/BNC, débits/encaissements).
  • Compte(s) bancaire(s) connecté(s) et à jour.
  • Justificatifs reçus pour la période.
  • Paramètres d'export vers la production cabinet.

3. Workflow côté collaborateur

  1. Réception : transactions bancaires + justificatifs (photographiés par le client).
  2. Contrôle de la pré-catégorisation : Tiime propose un compte/une TVA → vérifier et corriger.
  3. Association justificatif ↔ transaction : confirmer le rapprochement.
  4. Notes de frais : valider, rattacher la pièce, traiter le remboursement (425 / compte associé).
  5. Factures de vente (si émises via Tiime) : contrôler la TVA et l'imputation.
  6. Pièces manquantes : relancer ; transaction sans pièce → en attente + note.
  7. Export / tenue : exporter les écritures validées vers la production (ou poursuivre la tenue/TVA).

4. Points de vigilance

  • La catégorisation automatique n'est qu'une suggestion : la contrôler (compte + TVA).
  • Notes de frais : attention aux dépenses personnelles glissées (à signaler, pas de TVA).
  • TVA sur encaissements pour les prestataires de services.
  • Cohérence entre facturation Tiime et écritures de vente.

5. Erreurs fréquentes

  • Accepter la pré-catégorisation sans contrôle.
  • Note de frais sans justificatif valable.
  • Doublon (justificatif + facture).
  • Mauvais compte de remboursement de note de frais.

6. Bonnes pratiques offshore

  • Vider la file de transactions à traiter régulièrement.
  • Vérifier systématiquement compte + TVA proposés.
  • Documenter les anomalies ; relancer les pièces tôt.
  • Caler l'export avec le format attendu par le cabinet.

✅ Checklist Tiime (mois d'un dossier)

  • ☐ Transactions de la période toutes traitées
  • ☐ Pré-catégorisations contrôlées (compte + TVA)
  • ☐ Justificatifs associés
  • ☐ Notes de frais validées et rattachées
  • ☐ Factures de vente cohérentes
  • ☐ Pièces manquantes relancées
  • ☐ Export vers la production réalisé / TVA préparée

🧪 Cas pratique 22.1 — Note de frais

Le dirigeant photographie : (a) un repas client 48 € TTC, (b) un plein de carburant 70 € (VP), (c) des courses alimentaires personnelles 35 €.

✔️ Corrigé :

  • (a) repas client → 6257, TVA 20 % récupérable (justifié pro) ; remboursement via 425/455.
  • (b) carburant VP → 60622, TVA partiellement déductible (Module 5).
  • (c) courses personnelles → dépense personnelle : ne pas passer en charge société, signaler ; si payé par le compte pro, sortir en compte courant d'associé (455).
